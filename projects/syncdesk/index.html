Workspace data

One local workspace. Two connected desks.

SyncDesk and OpsDesk share one browser workspace. Data is not uploaded, so a workspace export is the recovery copy when site data is cleared or you change devices.

Local autosave
Every connection, field, formula, change, discrepancy, and linked OpsDesk item is saved in this browser.
Export workspace
Downloads one complete JSON file containing compatible data for both apps.
Import workspace
Merge the file with existing records or replace the browser workspace completely.

Restore workspace
