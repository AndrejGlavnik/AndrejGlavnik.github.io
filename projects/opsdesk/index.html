Before you start

Your workspace lives in this browser.

OpsDesk has no account or cloud database. Every edit is autosaved to this browser's site storage, but that copy can be erased.

Local autosave
Refreshes and normal browser restarts keep your work on this device.
What can remove it
Clearing website data, private browsing, browser cleanup, or moving to another browser or device.
Reliable continuation
Use Export report and choose Workspace backup, then import it later to merge with or replace a workspace.

Workspace backup is the complete restore file. Excel, CSV, and Markdown are reports for people, not restore files.

Workspace data

Local by design. Portable by backup.

OpsDesk and SyncDesk autosave to localStorage for this site. Nothing is uploaded, and there is no server copy to recover after browser data is cleared.

Export report
Choose Excel, CSV, or Markdown for reporting, or Workspace backup for a complete restorable copy of both apps.
Import workspace
Continue in this or another browser. Choose Merge to keep both workspaces or Replace to restore the file exactly.
Report formats
Useful for people, spreadsheets, and status sharing. They intentionally do not replace the complete workspace backup.

Best practice: export a Workspace backup after important changes and before clearing browser data or changing devices.
